# AudioDB

Audio DB is a backend API for storing and retrieving audiophile earphone information.

The project is built with Java Spring Boot, PostgreSQL, and AWS infrastructure. It focuses on providing structured and validated earphone data through RESTful APIs.

Current version goals:
- Store and retrieve earphone information accurately
- Support user authentication and authorization
- Allow admin moderation of user-submitted entries

Future goals:
- Community-driven earphone submissions
- Automated online data scraping
- Expanded audiophile database coverage


## Tech Stack

Backend:
- Java
- Spring Boot
- Spring Security
- JWT Authentication

Database:
- PostgreSQL

Infrastructure:
- AWS

Tools:
- Maven
- Git
- Postman

## Features

- RESTful API design
- User registration and login
- JWT-based authentication
- Secure password hashing
- Earphone CRUD operations
- Admin approval workflow
- PostgreSQL persistence
